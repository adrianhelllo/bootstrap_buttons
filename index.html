Buttons

Bootstrap buttons are available in every contextual colors and some extras.

These classes can be applied to a tags, buttons, and inputs of various types.

Link button

We can also create outlined buttons with similar color choices:

To customize the sizes of buttons, you can create large, medium (default) and small buttons.

To create full-width buttons, set the parent container to .d-grid.

.btn-block has been removed as of BS5.

You can also make it block-level with btn-block.

With multiple block level buttons, you can define the space between them with the gap-* class.

You can create active and disabled looking buttons. Active makes the button appears pressed, while disabled buttons are unclickable.
